# tgimg
适用于claw cloud部署的tgsearch、tgsou镜像

存储可选，挂载存储的好处就是使用命令行扫码后的session文件不会丢失，即使重启容器也不用重新获取session了。
tgsearch挂载目录建议/tmp

可以使用环境变量方式设置session，可以使用在线TG Session获取工具 https://tgs.252035.xyz
